# Vetor

Vetores so permitem guardar valores do mesmo tipo de dado

```rs
let v: Vec<i32> = Vec::new(); // vetor sem nenhum valor inserido, mas inferido ao tipo i32
let v = vec![1, 2, 3];//vec! macro para criação de vetore

```
